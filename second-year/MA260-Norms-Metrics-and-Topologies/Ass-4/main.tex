\question{1}

\begin{questionbody}
Let $X$ be a topological space and $A \subset X$. Suppose that for each $x \in A$ there is an open set $U$ containing $x$ such that $U \subset A$. Show that $A$ is open in $X$.
\end{questionbody}

Let $U_x$ be the associated open set for a particular $x$. Then $\cal U = \bigcup \{U_x : x \in A\}$ is an arbitrary union of open sets in a topological space, so is open. But $\cal U$ contains every point in $A$ and no points outside of $A$, since $U_x \subset A$ for all $x$. Therefore $\cal U = A$, so $A$ is open.

\hfill $\square$

% {{{ Q2
\question{2}

\begin{questionbody}
Let $f : S^1 \to \R$ be a continuous map. Show that there exists a point $x$ of $S^1$ such that $f(x) = f(-x)$. Here $S^1$ is the unit circle in $\R^2$ and $-x$ is the antipode of $x$.
\end{questionbody}

This is the 1-dimensional case of the Borsak--Ulam theorem. % chktex 8

Let $g : S^1 \to \R$ be defined by $g(x) = f(x) - f(-x)$. Then $g$ is continuous and since $S^1$ is connected, we can apply the intermediate value theorem. Either $g(x) = 0$ for all $x$, in which case we're done, or $g(x) > 0$ for some $x \in S^1$. In this case, $g(-x) = -g(x) < 0$, so $g(-x) < 0 < g(x)$. By the intermediate value theorem, there exists a $y \in S^1$ such that $g(y) = 0$, therefore $f(y) = f(-y)$.

\hfill $\square$

% {{{ Q3
\newquestion{3}

\begin{questionbody}
\begin{enumerate}[(a)]
\item Show that the product of two path connected spaces is path connected.

\item If $f : X \to Y$ is continuous and $X$ is path connected, is $f(X)$ necessarily path connected? Justify your answer.
\end{enumerate}
\end{questionbody}

\subsection{~} % 3.a

Let $X$ and $Y$ be path connected topological spaces, let $x_1, x_2 \in X$, and let $y_1, y_2 \in Y$. Let $\phi_X : [0, 1] \to X$ be a continuous path in $X$ from $x_1$ to $x_2$ and $\phi_Y : [0, 1] \to Y$ be a continuous path in $Y$ from $y_1$ to $y_2$.

Then the map $\phi_{X \times Y} : [0, 1] \to X \times Y$ defined by $\phi_{X \times Y}(t) = (\phi_X(t), \phi_Y(t))$ is a path from $(x_1, y_1)$ to $(x_2, y_2)$ in $X \times Y$ with the product topology. Since this is true for any $x_1$, $x_2$, $y_1$, and $y_2$, we conclude that $X \times Y$ is path connected.

\hfill $\square$

\subsection{~} % 3.b

Let $x_1, x_2 \in X$ and $\phi : [0, 1] \to X$ be a continuous path from $x_1$ to $x_2$. Since $f$ and $\phi$ are both continuous, $f \circ \phi : [0, 1] \to f(X)$ is also continuous and therefore defines a path from $f(x_1)$ to $f(x_2)$. This is true for any $x_1$ and $x_2$, so $f(X)$ is path connected.

\hfill $\square$

\newquestion{4}

\begin{questionbody}
Let $X$ be compact and $Y$ be Hausdorff. Show that if $f : X \to Y$ is continuous then $f$ is a closed map, so the image of every closed set is closed.
\end{questionbody}

Let $A \subset X$ be closed. By Theorem~8.17, a continuous image of a compact set is compact, so $f(A)$ is compact. By Theorem~8.13, any compact subset of a Hausdorff space is closed. Therefore $f(A)$ is closed for any closed $A \subset X$, so $f$ is a closed map.

\hfill $\square$
